# DIYAlbum
一个DIY的相册。

现在的任务 ： 应用Logo,

初步计划：只做本地数据

## 计划



### 1. 创建Album

####  1.1 添加图片

1. 添加网络图片
2. 添加本地图片
3. 设置图片位置


####  1.2 添加文字

1. 设置文字颜色
2. 设置文字大小
3. 设置文字位置


####  1.3 添加声音

1. 添加网络声音
2. 添加录音
3. 添加本地音乐

####  1.4 添加视频

1. 添加本地视频
2. 添加网络视频


### 2. 保存Album

#### 2.1 保存数据
1. 保存时间
2. 设置主题名称
3. 是否允许再次编辑


#### 2.3 草稿数据
1. 如果不许再次编辑，设置一个草稿

#### 2.2 再次编辑
1. 对数据进行反解析
