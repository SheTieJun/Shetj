# Android stuido 忽略文件 \#\#\#\#

打开Setting-Version Control-Ignored Files，添加忽略文件，我所知道的忽略文件如下：

1. idea文件夹
2. gradle文件夹
3. 所有的build文件夹
4. 所有的.iml文件
5. local.properties文件。
